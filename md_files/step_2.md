# Passo 2/7 — Criar suas Playlists Iniciais

**Instrução**  
No mesmo chat do LLM que você iniciou no Passo 1, gere de 3 a 5 playlists alinhadas aos seus pilares.

Use o prompt abaixo:

```
Com base nos meus pilares, gere de 3 a 5 playlists para o meu canal, cada uma com:
- nome curto
- uma mini descrição de uma linha
As playlists devem refletir meu posicionamento e meu objetivo definidos anteriormente.
```

**Após executar no LLM**  
Volte para a plataforma e clique em **Próximo passo**.
