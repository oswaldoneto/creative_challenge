# Passo 4/7 — Criar a Mini-Identidade Provisória

**Instrução**  
No mesmo chat do LLM, gere uma mini-identidade provisória para o canal.

Use o prompt abaixo:

```
Sugira três opções de mini-identidade provisória contendo:
- três cores principais
- uma regra simples de legibilidade
- um estilo visual sugerido
As opções devem ser coerentes com meus pilares e minhas playlists.
```

**Após executar no LLM**  
Volte à plataforma e clique em **Próximo passo**.
