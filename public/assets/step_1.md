
# Passo 1/7 — Iniciar o Contexto com o LLM

**Instrução**  
Abra um único chat no LLM de sua preferência.  
Você usará esse mesmo chat até o final do exercício para manter o contexto.

Copie o prompt abaixo, ajuste os placeholders e execute no LLM.

```
Você será meu assistente no Setup IA Ready.
Use este mesmo chat até o final para manter o contexto.

Meus pilares são: [INSERIR PILARES].
Meu posicionamento como creator é: [DESCREVER].
Meu objetivo com este setup é: [DESCREVER].

Confirme que entendeu e indique que está pronto para o próximo passo.
```

**Como ajustar os placeholders**  
- [INSERIR PILARES] — liste seus 3 ou 4 pilares.  
- [DESCREVER] — tom, estilo ou área do canal.  
- [DESCREVER] — objetivo em 1 linha.

**Após executar no LLM**  
Volte à plataforma e clique em **Próximo passo**.
